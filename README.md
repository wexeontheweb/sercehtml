# 💓 Serce HTML

---

## ✨ O projekcie

Kod aplikacji, która pokazuje na początku *"Ej bo ogólnie..."*. a po kilku sekundach ekranie zaczyna pojawiać się napis *"Kocham cię"*.


## 🚀 Jak uruchomić?

1. Pobierz lub skopiuj kod z pliku `index.html`.
2. Zapisz plik na swoim komputerze.
3. Kliknij na niego dwukrotnie, aby otworzyć go w dowolnej przeglądarce internetowej.
